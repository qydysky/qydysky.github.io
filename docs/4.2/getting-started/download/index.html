    下载
    
          
    下载Bootstrap以获取已编译的CSS和JavaScript，源代码，或将其包含在您最喜欢的包管理器中，如npm，RubyGems等。
    

          
    已编译的CSS和JS
    


    下载Bootstrap v4.2.1即用型编译代码，轻松在您的项目中使用，其中包括：
    


      
  
    • 已编译和压缩的CSS包（参见CSS文件比较）
      • 
  
    • 已编译和压缩的JavaScript的插件
      • 

    


    这不包括文档，源文件或任何可选的JavaScript的依赖项（jQuery的和Popper.js）。
    


    下载
    


    源文件
    


    通过下载我们的Sass的JavaScript和文档文件，使用您自己的管道编译引导。此选项需要一些额外的工具：
    


      
  
    • 用于编译CSS的Sass编译器（支持Libsass或Ruby Sass）。
      • 
  
    • 用于CSS的Autoprefixer
      • 

    


    如果您需要构建工具，它们包含在开发引导及其文档中，但它们可能不适合您的目的。
    


    下载源代码
    


    BootstrapCDN
    


    使用BootstrapCDN跳过下载，将引导编译的CSS和JS的缓存版本直接提供给您的项目。
    


    


    如果您正在使用我们编译的JavaScript，请不要忘记在它之前添加jQuery和Popper.js的CDN版本。
    


    


    包管理员
    


    使用一些最受欢迎的软件包管理器，将Bootstrap的源代码插入到几乎所有项目中。无论包管理器如何，Bootstrap都需要一个Sass编译器和Autoprefixer来进行与我们官方编译版本相匹配的设置。
